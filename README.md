## Slack.com Setup Steps


## Environment Variables

Create a file called `.env` with the following variables:
```
SLACK_SIGNING_SECRET=<KEY_HERE>
SLACK_BOT_TOKEN=<KEY_HERE>
SLACK_APP_TOKEN=<KEY_HERE>
```
